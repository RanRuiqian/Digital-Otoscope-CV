# Digital Otoscope CV

A low-cost digital otoscope prototype designed for clinical teaching. This is an industry-university research project developed at Monash University Malaysia.

## Hardware Integration
- 3.9mm micro-camera module
- Custom mounting and wiring for clinical application

## Software & Computer Vision
- Python
- OpenCV

## Current Status (Version 1)
Currently developing the initial prototype. The main focus is on:
1. Resolving hardware communication stability.
2. Processing real-time video streams (noise reduction and edge enhancement).
3. Correcting imaging distortion in confined spaces.
